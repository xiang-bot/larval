变量分配与展示
三种方式
view('模板'，数组);
view('模板')->with(数组);
view('模板')->with(名称，值)->with(名称,值);
展示方法：
	public function view1(){
		$date = date('Y-m-d',time());
		$day = '日';
		return view('view1',['date'=>$date,'day'=>$day]);
	}
视图中：
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
{{$day}}<br />
{{$date}}
</body>
</html>
显示：
日
2019-07-25

使用compact()函数，php内置函数与框架无关。作用是用于打包数组
compact('变量名1','变量名2','变量名3',...);将变量的值自动套上去

	public function view1(){
		$date = date('Y-m-d',time());
		$day = '日';
		$city = 'Wuhan';
		$province = 'Hubei';
		$location = array('city','province');
		return view('view1',compact('date','day','city','province'));
	}
------------------------------
<body>
{{$day}}<br />
{{$date}}
{{$city}}
{{$province}}
</body>
